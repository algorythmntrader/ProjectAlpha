Welcome to ProjectAlpha!

ProjectAlpha a capable high performance trading software with strong focus on automation.
It provides a complete tool chain for design, code, test and RUN your trading algorithm's.
The trading strategy deployment has collocation in mind - so all trading strategies are hosted on server side!
For sure there is a client to manage and monitor the deployed trading strategies.
If you want to try out, feel free to download the latest release!

Don't use it in a live production environment! Use this software on your own risk!
